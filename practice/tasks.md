# 🛠️ Практичні завдання та лабораторні роботи

## 🛑 Лабораторна робота №1: Дослідження процесів у вашій системі (Базовий рівень)

**Мета:** Навчитися аналізувати використання ресурсів ОС Windows або Linux.

### Хід роботи:
1. Відкрийте **Диспетчер завдань** (Task Manager) за допомогою комбінації клавіш `Ctrl + Shift + Esc` (або `htop` у терміналі Linux).
2. Перейдіть на вкладку "Процеси". Знайдіть програму, яка споживає найбільше оперативної пам'яті.
3. Перейдіть на вкладку "Продуктивність" (Performance). Зафіксуйте поточний відсоток завантаження CPU (процесора) та RAM.
4. Запустіть важку програму (гру або важке відео в 4K) і запишіть, як змінилися графіки використання ресурсів.
5. Зробіть скріншоти та оформіть короткий звіт у вашому зошиті/документі.

## 🐚 Лабораторна робота №2: Керування файловою системою через CLI (Просунутий рівень)

**Мета:** Освоїти базові команди командного рядка (CLI) для навігації та створення структури папок.

### Хід роботи:
1. Запустіть Командний рядок (`cmd` в Windows) або Термінал (в Linux/macOS).
2. Дізнайтеся, в якій папці ви перебуваєте, за допомогою команди:
   * Windows: `echo %cd%`
   * Linux/macOS: `pwd`
3. Створіть нову папку з назвою `OS_Practice` за допомогою команди: `mkdir OS_Practice`
4. Перейдіть у створену папку: `cd OS_Practice`
5. Створіть у ній текстовий файл із вашим ім'ям:
   * Windows: `echo Hello Student > name.txt`
   * Linux/macOS: `echo "Hello Student" > name.txt`
6. Перегляньте вміст папки командою `dir` (Windows) або `ls` (Linux).
