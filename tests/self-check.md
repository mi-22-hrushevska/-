# 🧪 Тести для самоперевірки知識

Перевірте, як добре ви засвоїли принципи роботи операційних систем.

### ❓ Питання 1 (Одиничний вибір)
**Який компонент операційної системи безпосередньо взаємодіє з апаратним забезпеченням комп'ютера?**
* А) Командна оболонка (Shell)
* Б) Ядро (Kernel)
* В) Диспетчер завдань
* Г) Антивірус

> **Правильна відповідь:** Б. Ядро є серцем ОС і управляє залізом.

### ❓ Питання 2 (Множинний вибір)
**Які з перерахованих назв є файловими системами? (Виберіть декілька варіантів):**
* [ ] А) NTFS
* [ ] Б) HTTP
* [ ] В) ext4
* [ ] Г) RAM
* [ ] Д) APFS

> **Правильна відповідь:** А, В, Д. (HTTP — це мережевий протокол, RAM — тип пам'яті).

### ❓ Питання 3 (Встановлення відповідності)
**Зіставте аналогію з реального життя та концепцію ОС:**

| Аналогія | Концепція ОС |
| :--- | :--- |
| 1. Перекладач з іноземної мови | А) Віртуальна пам'ять |
| 2. Кухар, що готує страву | Б) Поток (Thread) |
| 3. Оренда додаткового складу, коли свій заповнений | В) Драйвер пристрою |

> **Правильна відповідь:** 1-В, 2-Б, 3-А.
