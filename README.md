# Навчальний проект: Принципи роботи операційних систем

Цей репозиторій містить комплекс навчальних матеріалів з інформатики для учнів 10–11 класів, присвячений архітектурі, функціям та принципам функціонування сучасних операційних систем (ОС).

* **Цільова аудиторія:** Учні 10–11 класів (профільний рівень).
* **Мета матеріалів:** Сформувати системне розуміння того, як ОС керує апаратними ресурсами комп'ютера та забезпечує взаємодію з користувачем і програмами.
