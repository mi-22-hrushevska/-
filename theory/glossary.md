# 📖 Глосарій основних термінів

| Термін | Визначення |
| :--- | :--- |
| **Ядро ОС (Kernel)** | Центральна частина операційної системи, яка керує процесором, пам'яттю та пристроями введення-виведення. |
| **Драйвер** | Спеціальна програма, яка дозволяє ОС взаємодіяти з конкретним апаратним пристроєм (наприклад, принтером чи відеокартою). |
| **Процес** | Екземпляр програми, що виконується, якому ОС виділила ізольовані системні ресурси. |
| **Переривання (Interrupt)** | Сигнал від апаратного чи програмного забезпечення, який змушує процесор тимчасово призупинити поточну роботу для виконання термінового завдання. |
| **Файлова система** | Порядок, що визначає спосіб організації, зберігання та іменування даних на носіях інформації. |
| **Планувальник завдань** | Компонент ядра ОС, який вирішує, який процес і протягом якого часу буде виконуватися процесором. |
